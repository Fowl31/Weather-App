Weather App

A simple weather application that fetches real-time weather data using the OpenWeather API.

## Features

- Search weather by city
- Current temperature
- Weather description
- Humidity
- Wind speed
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript
- OpenWeather API
